# baymax-completions
This is an autocomplete provider for [Baymax](https://airtable.com/styleguide), Airtable's CSS framework, to be used with the [`autocomplete-plus`](https://github.com/atom/autocomplete-plus) package. In any `className` prop or [`classNames`](https://github.com/JedWatson/classnames) function, this plugin will show you suggested helper classes as you type. Inspired by BuzzFeed's [`solid-completions`](https://github.com/buzzfeed/solid-completions).

## Installation
```
$ cd ~/.atom/packages
$ ln -s ~/blocks-sdk/less/baymax-completions
```
Whenever the stylesheets are updated, run `gulp` to rebuild the class definitions.
